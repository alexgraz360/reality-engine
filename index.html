About Reality Engine

An open platform that hosts swappable reality modes — astronomy, physics, and whatever comes next — sharing one shell, one sensor pipeline, and (soon) one AI companion. Phone-first now, glasses-ready by design.

Open-source credits

  • Pendulum period method inspired by phyphox (GPLv3 — method referenced, no code reused).
  • Built with plain web standards: ES modules, Canvas, DeviceMotion, getUserMedia. No frameworks, no build step, no CDN.
  • Sibling apps: astronomy-glasses · physics-glasses (migrating in).
